# 🚀 Professional Portfolio | React & Tailwind CSS

Welcome to my personal portfolio repository. This project is a modern, component-driven web application designed to showcase my journey as a **Computer Science student** and **Full-Stack Developer**. 

Built with a focus on clean code, performance, and a minimalist aesthetic, this site serves as a central hub for my projects and technical skills.

## 🛠️ Tech Stack

* **Core:** [React.js](https://reactjs.org/) (Functional Components & Hooks)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Utility-first styling)
* **Iconography:** [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
* **Typography:** [Poppins](https://fonts.google.com/specimen/Poppins), [Inter](https://fonts.google.com/specimen/Inter), and [Outfit](https://fonts.google.com/specimen/Outfit)
* **Deployment:** [Vercel / Netlify]

## ✨ Key Features

* **Component-Based Architecture:** Modular UI elements for high reusability and scalability.
* **Fully Responsive:** Optimized for all screen sizes, from mobile devices to large desktops.
* **Optimized Performance:** Fast load times using efficient React patterns and Tailwind's JIT engine.
* **Modern UI/UX:** Smooth transitions and accessible icons for an enhanced user experience.
